# الگوریتم‌های مهم هوش مصنوعی
## پیاده‌سازی الگوریتم‌های یادگیری ماشین، یادگیری عمیق و یادگیری تقویتی با Python

این مخزن شامل تمامی کدها، مثال‌ها، پروژه‌های کاربردی و فایل‌های اجرایی کتاب 
«الگوریتم‌های مهم هوش مصنوعی» است.

هدف این مخزن، فراهم کردن محیطی برای یادگیری عملی الگوریتم‌های هوش مصنوعی 
و ارتباط میان مبانی نظری، ریاضیات، پیاده‌سازی Python و کاربردهای واقعی است.
## درباره کتاب

کتاب «الگوریتم‌های مهم هوش مصنوعی» با هدف آموزش جامع الگوریتم‌های پرکاربرد 
در حوزه‌های زیر تألیف شده است:

- یادگیری ماشین (Machine Learning)
- یادگیری عمیق (Deep Learning)
- یادگیری تقویتی (Reinforcement Learning)
- الگوریتم‌های بهینه‌سازی و تکاملی

در هر فصل، علاوه بر تشریح مبانی نظری و ریاضی الگوریتم‌ها، 
یک مثال عددی، پیاده‌سازی Python، تحلیل ابرپارامترها، 
معیارهای ارزیابی و یک مطالعه موردی کاربردی ارائه شده است
## مخاطبان

این مجموعه برای گروه‌های زیر طراحی شده است:

- دانشجویان مهندسی کامپیوتر
- دانشجویان هوش مصنوعی
- دانشجویان علوم داده
- پژوهشگران حوزه یادگیری ماشین
- مهندسان علاقه‌مند به کاربردهای هوش مصنوعی
